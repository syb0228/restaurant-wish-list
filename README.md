# restaurant-wish-list

## SKILLS
- **Backend**
    - `JDK 11 (JAVA)`
    - `Spring-boot`
- **Frontend**
    - `bootstrap framework`
- **IDE** : `IntelliJ`
    - **build tool** : `gradle`
- **Open Source** : `Naver API`

## 기능 요구 사항
- 맛집 검색
- 맛집 위시리스트 추가
- 맛집 위시리스트 삭제
- 맛집 방문 추가

## 구현 방식
- 네이버 지역 / 이미지 검색 API 활용
- Memory CRUD DB 개발
- REST API 개발
- 간단한 Front-end 디자인
